Este projeto foi desenvolvido usando Spring Boot e diversas bibliotecas essenciais para facilitar a criação de uma aplicação web robusta e escalável.

Tecnologias Utilizadas

Spring Boot Starter Web

Proporciona um ambiente rápido para desenvolver e configurar aplicações web, simplificando a configuração do servidor embutido e tornando o processo de criação mais intuitivo.

Spring Boot Starter Thymeleaf

Permite a integração com o Thymeleaf, um mecanismo de template moderno para HTML, facilitando a construção de páginas dinâmicas e interativas no frontend.

Spring Boot Starter Data JPA

Fornece uma integração simplificada com o JPA (Java Persistence API), permitindo o mapeamento e manipulação de dados em bancos de dados de forma eficiente.

H2 Database

Um banco de dados em memória leve e de fácil configuração, ideal para desenvolvimento e testes locais sem a necessidade de um servidor de banco de dados externo.
